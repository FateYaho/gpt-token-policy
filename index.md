## Privacy Policy

This Chrome extension ("Extension") does not collect, store, or transmit any personal data to external servers. All data processed by the Extension remains strictly within the user's local browser environment.

### Data Access
The Extension accesses text displayed on ChatGPT-related websites (chat.openai.com and chatgpt.com) solely for the purpose of calculating token usage. The accessed text is not transferred, logged, or shared with any external service.

### Data Storage
Some optional user settings (such as model selection, preferences, and token usage logs) may be saved locally using Chrome's storage API. This information never leaves the user's device.

### No Third-Party Sharing
The Extension does not sell, share, or transfer any user data to third parties.

### No External Servers
The Extension does not send any data outside the local browser. No analytics, tracking, or remote processing is performed.

### Permissions
The permissions requested by the Extension (scripting, activeTab, tabs, storage, downloads, alarms, etc.) are used strictly for operating token calculation and UI functionality within the allowed domain.
